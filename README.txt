This program takes in 4 postal code as inputs, and computes the shortest path 
leaving from home, going through all the given points in map, and going back home.
The output provides the user a map of the route, the sequence of the route, 
as well as the total distance to be travelled according to the route.

Input format: UPPER letter from 'A' to 'F' followed by number from '1' to '6', all four points have to be given at all times
Output: shortest path
